# IBRX11 Data Analysis (Research Sample)

This repository contains the official sample code used in the academic paper:

https://papers.ssrn.com/sol3/papers.cfm?abstract_id=6300840

---

## 📖 About the Project

This project provides the original implementation of the analyses and statistical tests presented in the paper.

It allows the research methodology and results to be explored directly through code.

---

## Academic Context

The paper investigates whether the largest components of an index are sufficient to explain its overall performance.

This repository contains the sample analysis used in the study.

---

## What this Project Includes

* Data preprocessing
* Statistical analysis
* Evaluation of index components
* Chart generation

---

## Project Structure

Sample_Ibrx11/

* analise.py → main analysis script
* charts/ → generated charts
* cotacoes_ibrx11.xlsx → dataset
* requirements.txt → dependencies

---

## Purpose

To provide transparency and reproducibility for the analysis presented in the paper.
